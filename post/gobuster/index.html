Gobuster

What is Gobuster?

Gobuster is a Linux command line tool used to view all hidden pages of a website. Some websites will leave admin panels unprotected, leaving them vulnerable for us to attack. For this, will will be using a TryHackMe website called fakebank.com, so lets get into it. Note, this will not work if you use a general Linux Terminal, you must use the TryHackMe room.

Initiating the attack

To begin the attack, we are going to open the Terminal and type in the follow command:

gobuster -u http://fakebank.com -w wordlist.txt dir

-u is telling Gobuster that http://fakebank.com is the target URL, and -w is telling it where to put the output,

As we can see from this, we have retrieved a /bank-transfer page which, to me, sounds like the place where I would be transferring money. From this, we can transfer money from one account to another, which in a real-life situation, is something a bank does not want happening.

This was just an example as to what you can do with Gobuster. You can try it out with pretty much any website you have the consent of, because remember lads, theres only one type of hacker, and it’s the ethical one. It’s illegal to hack someones personal web application, device, server, cloud network, etc without their permission, so make sure you have their full consent.

Furthermore, Gobuster has already been used for some pretty large attacks from earlier years, but with the advancement in web protection and that pretty much every website makes you log in or sign up to their service before you can view any page, it has become ever so hard to use Gobuster in a way to hack into a web service as easily as 1 command. Maybe in the future, but not now, so we will have to wait and see for the advancement in Gobuster and it’s surrounding technology.
